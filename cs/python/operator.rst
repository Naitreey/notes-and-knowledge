content
=======

- 各种 python 标准 operators 的 functional form.

- 一些通用的操作的函数形式, 包含:

  * ``countOf()``

  * ``indexOf()``

- 一些特别的 convenience callables.

  * ``attrgetter``

  * ``itemgetter``

  * ``methodcaller``

usage
=====

可用于需要 functional form 来方便地一般化某些操作或函数.
例如与一些使用 functional programming, lambda expression 的场景重复.
